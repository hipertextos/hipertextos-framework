Hipertextos Framework

CSS Animate

Hipertextos package includes Dan Eden's CSS animate to work with CSS animations. Some features are added to the original version

  • Author: Dan Eden.
  • Web: daneden.me/animate/.
  • Description: "animate.css is a bunch of cool, fun, and cross-browser animations for you to use in your projects. Great for emphasis, home pages, sliders, and general just-add-water-awesomeness".
  • License: Visual Idiot.
Use
<link href="../css/animate.css" rel="stylesheet" media="screen">

See the animations page to know about animations in hipertextos framework.

Google prettify

Hipertextos package includes jQuery Google prettify to display code.

  • Author: Mike Samuel.
  • Web: Google code prettify.
  • Description: "A Javascript module and CSS file that allows syntax highlighting of source code snippets in an html page".
  • License: Apache license 2.0.
Use
<link href="../css/google.prettify.css" rel="stylesheet" media="screen">
<script src="../js/google.prettify.js"></script>
HTML Markup
<pre class="prettyprint linenums"></pre>

Easy Tabs

Hipertextos package includes jQuery Easytabs

  • Author: Steve Schwartz (JangoSteve).
  • Web: jQuery Easytabs.
  • Description: "EasyTabs is a lightweight jQuery plugin to provide full tab functionality, while leaving the styling up to you.".
  • License: MIT and GPL.
Use
<script src="../js/jquery.easytabs.js"></script>

Jquery easy tabs works with the same HTML markup that hipertextos.js, adding effects and multiple instances.

Tocify

Hipertextos package includes jQuery Tocify

  • Author: Greg Franko.
  • Web: jQuery Tocify.
  • Description: "A jQuery Table of Contents plugin that can be themed with Twitter Bootstrap or jQueryUI".
  • License: MIT.
Use
<script src="../js/jquery.ui.widget.js"></script>
<script src="../js/jquery.tocify.js"></script>
<script>
	$(function() {
		// Calls the toc-side and hide it
		$(".page").removeClass("pagedisplaced");
		//Calls the tocify method on your HTML div.
		$("#toc").tocify({ selectors: "h3" });
		$("#toc").animate({width: 'toggle'}, 500);
	});  
	$("#tocing").click(function(){
		$("#toc").animate({width: 'toggle'}, 500);
		$('#toc').toggleClass('animated bounceInLeft');
		$(".page").toggleClass("pagedisplaced");
		$(".toc-button").toggleClass("expanded");
	});
</script>
HTML Markup
<!-- TOC -->
<div class="toc-button no-js" id="tocing"></div>
<div class="toc-side no-js" id="toc"></div>
