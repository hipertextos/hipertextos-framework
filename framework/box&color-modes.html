Box & color modes

Hipertextos Framework

The Box Model

The "box" class creates a reusable and adapatative fluid container to use in any context.

Generic Box

The box internal structure are divided in header, .corpus, and footer. The icon button top right positioned is used to close modals boxes. Buttons inside footer are floating right.
