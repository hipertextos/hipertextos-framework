Hipertextos Framework

Globals

Hipertextos CSS uses a short CSS Reset inspired by Eric Meyer's reset.

Hipertextos CSS uses a 8px baseline with 16px default font-size and 24px default line-height. All elements are aligned in a 8px vertical grid.

Hipertextos CSS uses a sans-serif font stack by default: "Helvetica Neue", Helvetica, Arial, sans-serif.
