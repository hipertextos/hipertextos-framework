Icon fonts: the smart solution

Font icons are a great solution to manage icons in web developement. Font icons are scalable, colorable and lightweight. Use cool font icons is easily with online services as Icomoon. By default, hipertextos.css includes 128 font icons frequently used in web developement. Font icons included can be used in two simple ways, as objects or inside CSS elements (like buttons or lists).

If you want work with graphic icons, you may take a look at the hipertextos icons library.

Icon as objects

Use font icons as objects with the "data-icon" attribute.

 <div aria-hidden="true" data-icon="ICONIDHERE"></div>

Icon sizes

Set the icon size easily with icon-size classes.

 <div class="ic-SIZE"></div>

Icons inline

Use font icons inside buttons, lists or inline elements adding the "icon" class in the HTML elements.
