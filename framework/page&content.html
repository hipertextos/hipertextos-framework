Page & content

Hipertextos Framework

Page

"Page" class works like a global container. Modals, Table of Content and notifications should be keep outside "page".

Content

"Content" class works like a page wrapper with fixed-width, centering the content and making it responsive.

Content-fluid

"Content-fluid" class works like a fluid page wrapper, expanding all content to full screen device.

Page sections

Page are divided in header, content and footer.

Reusable content classes

Content and content-fluid classes are reusable to center any content in a page and making it responsive.

Important: Header and footer are reusable inside article, section, aside or any box class. Otherwise content-fluid class must be used inside this elements.
