Hipertextos Framework

One script, a lot of features

Hipertextos framework includes hipertextos.js script to work with jQuery. This script add useful features to your project with less than 1000 lines of code. To use this script in your web projects just must be include it.

<script src="../js/jquery.js"></script>
<script src="../js/hipertextos.js"></script>

Fallbacks

HTML5shim is used by default to add old browser support. Using font icons, icomoon-lte-ie7.js is needed to add support to iexplorer 6 and 7. In all cases, Javascript fallbacks are recommended.

<!--[if lt IE 9]><script src="http://html5shim.googlecode.com/svn/trunk/html5.js"></script><![endif]-->
<!--[if lt IE 7]><script src="../js/icomoon-lte-ie7.js"></script><![endif]-->
<!-- If JS disabled --><noscript><link href="../css/hipertextos-noscript.css" rel="stylesheet" media="screen"></noscript>

Modals

A modified version of Lean Modal is included in hipertextos.js. With this script is easy create custom modals in combination with "box" color modes and CSS3 animations. View more examples.

Demo

Launch modal

HTML Markup

<div class="modal box animated bounce" id="modal">
	<a class="icon-button" href="#">X</a>
	<header><h4>Modal title</h4></header>
	<div class="corpus"><p>This is a Modal</p></div>
	<footer>
		<p>This is a modal footer.</p>
	</footer>
</div>

JS

<script>
	$(function() {
		// Modals
		$('a[rel*=modal]').leanModal({ top : 200, overlay : 0.6, closeButton: ".icon-button" });
	});  
</script>

Tabs

Hipertextos.js includes a lightweight tabs contructor with 4 position styles (left, right, top and bottom). The included script only works with a single instance of tabs. This is enough for many web projects. Include the jquery.easytabs.js plugin in your page to create multiple tabs instances and add cool effects to your tabs with the same markup.

Table of Content (TOC)

You can add a sidebar with a TOC with this small javascript in your page. You can load a dynamic index of content inside TOC sidebar with the plugin jquery.tocify.js, used in this page. The scripts needed are included in the complete package.

Demo

You can see a demo of TOC sidebar in the left section of this page.

HTML Markup

<!-- TOC -->
<div class="toc-button no-js" id="tocing"></div>
<div class="toc-side no-js" id="toc"></div>

JS

$(function() {
	// Calls the toc-side and hide it
	$(".page").removeClass("pagedisplaced");
	//Calls the tocify method on your HTML div.
	$("#toc").animate({width: 'toggle'}, 500);
});  
$("#tocing").click(function(){
	$("#toc").animate({width: 'toggle'}, 500);
	$('#toc').toggleClass('animated bounceInLeft');
	$(".page").toggleClass("pagedisplaced");
	$(".toc-button").toggleClass("expanded");
});

JS to use with jquery.tocify

<script src="../js/jquery.ui.widget.js"></script>
<script src="../js/jquery.tocify.js"></script>
<script src="../js/google.prettify.js"></script>
<script>
	$(function() {
		// Calls the toc-side and hide it
		$(".page").removeClass("pagedisplaced");
		//Calls the tocify method on your HTML div.
		$("#toc").tocify({ selectors: "h3" });
		$("#toc").animate({width: 'toggle'}, 500);
	});  
	$("#tocing").click(function(){
		$("#toc").animate({width: 'toggle'}, 500);
		$('#toc').toggleClass('animated bounceInLeft');
		$(".page").toggleClass("pagedisplaced");
		$(".toc-button").toggleClass("expanded");
	});
</script>
